# td-audio-player

Final Build of TD Audio Player

## Tech Stack

- Node.js
- TypeScript
- Express

## Features

- Basic healthcheck endpoint
- Ready for environment variables
- Simple structure to extend

## Getting Started

1. Clone the repo:

   ```bash
   git clone <repo_url>
   cd td-audio-player
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Create a .env file if needed and set your environment variables.

4. Run in development:

   ```bash
   npm run dev
   ```

5. Build and run in production:

   ```bash
   npm run build
   npm start
   ```
